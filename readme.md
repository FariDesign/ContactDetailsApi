# Contact info api!

### Request

-  Method: POST
-   URL: `https://localhost:5001/api/contactinfo`
-   Headers:
-   Content-Type: application/json
-   Body: raw JSON

    {
    "firstname": "John",
    "lastname": "Doe",
    "email": "john.doe@example.com",
    "gender": "Male",
    "title": "Mr."
    }
